## Deskripsi Project
Website portfolio pribadi fullstack menggunakan Express.js (backend) dan Vue.js (frontend) dengan database SQLite.

## Cara Menjalankan
1. Clone repository ini
2. Install dependencies: `npm install`
3. Jalankan server: `node backend/app.js`
4. Buka browser: `http://localhost:3008`
